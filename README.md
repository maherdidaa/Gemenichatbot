# Facebook Messenger Automation Platform 🚀

An enterprise-grade, high-performance SaaS platform built to automate Facebook Messenger interactions, manage subscribers, run broadcast campaigns, and orchestrate visual chatbot flows.

## 🛠️ Tech Stack

- **Frontend**: Next.js (TypeScript, TailwindCSS, Radix UI/Shadcn, React Flow)
- **Backend**: NestJS, Socket.io (Realtime messaging)
- **Database**: PostgreSQL (via Prisma ORM)
- **Queue System**: Redis + BullMQ (For high-throughput Facebook Webhook processing)
- **Containerization**: Docker & Docker Compose
